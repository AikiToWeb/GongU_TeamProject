# 공간 대여 사이트 팀 프로젝트

## 주요 기능
 1. 상품 목록 구현
 2. 상품 상세 보기 구현
 3. 예약 기능 구현
 4. 게시판 게시글 등록, 수정, 삭제 구현
 5. 회원 가입, 로그인 기능 구현

- JSP MVC패턴 모델2 구조로 제작하였습니다.
- 상품 목록 구현시 상품정보, 지역 대분류, 지역 소분류를 join을 사용하였습니다.
- DB로는 MySql을 사용하였습니다.
- HTML, CSS, JavaScript는 bootstrap을 이용하였습니다.
